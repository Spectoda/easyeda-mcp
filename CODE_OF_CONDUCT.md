# Code of Conduct

Be kind, precise, and constructive.

This project sits at the intersection of software, electronics, manufacturing,
and AI agents. Mistakes can be expensive, so review feedback should be direct
without becoming hostile.

Expected behavior:

- explain risks clearly;
- credit prior work;
- keep private data private;
- assume good intent while verifying technical claims;
- avoid personal attacks, harassment, or discriminatory language.

Maintainers may remove comments, issues, pull requests, or contributors that
make the project unsafe or unwelcoming.
